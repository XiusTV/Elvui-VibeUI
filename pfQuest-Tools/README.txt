================================================================================
                         pfQuest Merger Tool
       Automatically captures WoW quest data and makes it permanent
================================================================================

RECOMMENDED: dist\pfQuest Merger Tool.exe
(No Python required! Graphical interface!)

ALTERNATIVE: Merger via CMD\RUN_MERGER_HERE.bat
(Requires Python 3.7+, command-line interface)

================================================================================

QUICK START:

1. Go to: dist\
2. Run: pfQuest Merger Tool.exe
3. Click Settings, set paths, click Save
4. Click "Start Monitoring"
5. Play WoW - it captures quests automatically!

When you close WoW, it saves everything.

================================================================================

FOLDERS:

dist\              - .exe version (USERS)
Merger via EXE\    - Build tools (DEVELOPERS)
Merger via CMD\    - Python version (ADVANCED)
