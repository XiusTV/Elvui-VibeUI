DESCRIPTION (Detailed - for About section):
============================================
pfQuest Enhanced is a comprehensive quest helper addon for World of Warcraft 3.3.5 (WotLK) with an automated quest capture system. Shows 10,000+ quests on map/minimap with smart routing, and automatically captures new quest data as you play. Includes standalone merger tool (.exe) that makes captured quests permanent and shareable. No manual database editing required!
